# PRD — 运营管理

| 属性 | 内容 |
|------|------|
| 模块 | 运营管理 |
| 页面 | 7 页（含 2 详情/编辑） |
| 优先级 | P1 |
| 版本 | V1.0 |

---

## 1. 模块概述

覆盖财务报销、经营分析、人事审批、薪资发放与员工档案，支撑组织内部运营管控。

---

## 2. 页面清单

| 页面 | 文件 | 说明 |
|------|------|------|
| 财务报销 | payment-request.html | 报销申请与审批 |
| 经营报表 | financial-reports.html | 收入利润图表 |
| 人事管理 | hr-request.html | 人事类审批 |
| 薪资管理 | salary-management.html | 工资条确认 |
| 员工管理 | department-employees.html | 部门树+员工列表 |
| 员工档案 | employee-profile.html | 详情 |
| 员工信息编辑 | employee-edit.html | 编辑 |

---

## 3. 财务报销（payment-request.html）

### 3.1 筛选

报销类型、审批状态（未审批/审批中/已通过/未通过/已支付）

### 3.2 Tab

我提交的 / 待我处理 / 已处理 / 抄送我的

### 3.3 列表字段

报销单号、报销类型、报销金额、申请时间、审批状态、操作

### 3.4 行操作（随 Tab/状态）

| 场景 | 操作 |
|------|------|
| 我提交-未审批 | 撤销 |
| 待我处理 | 审批 |
| 已通过待支付 | 确认支付 |
| 全部 | 查看详情 |

### 3.5 弹窗

| 弹窗 | 用途 |
|------|------|
| modal-apply | 新建报销（类型、金额、事由、附件、审批流程预览） |
| modal-detail | 审批详情（流程时间轴、通过/拒绝/撤销） |
| modal-confirm-pay | 确认支付 |

### 3.6 审批流程

部门审批 → 经理审批 → 财务支付

---

## 4. 经营报表（financial-reports.html）

### 4.1 筛选

统计周期（月份区间）、所属部门、业务类型

### 4.2 Tab

| Tab | 图表 | 说明 |
|-----|------|------|
| 收入与利润统计 | chart-profit | 组合图：收入柱+利润线 |
| 收入月度统计 | chart-monthly | 堆叠柱+合计折线 |

### 4.3 数据展示

- 概览 KPI 卡片（收入、成本、利润、利润率）
- Chart.js 交互图表
- Tab 切换时 resize 图表

---

## 5. 人事管理（hr-request.html）

> 页面标题、面包屑、按钮文案统一为「人事管理」。

### 5.1 申请类型 Tab

全部 / 招聘申请 / 转正申请 / 离职申请 / 调岗申请

### 5.2 列表 Tab

我提交的 / 待我处理 / 已处理 / 抄送我的

### 5.3 筛选

申请类型、审批状态、关键词

### 5.4 列表字段

申请单号、申请类型、申请人、申请时间、审批状态、操作

### 5.5 弹窗

| 弹窗 | 用途 |
|------|------|
| modal-apply | 新建申请（类型动态表单：招聘岗位/转正员工/离职原因等） |
| modal-detail | 审批详情（通过/拒绝按钮） |

### 5.6 审批流程

- 招聘：部门审批 → 人事审批
- 其他：部门审批 → 经理审批

---

## 6. 薪资管理（salary-management.html）

### 6.1 筛选

薪资月份、所属部门、员工姓名

### 6.2 列表字段

员工姓名、部门、岗位、应发、扣款、实发薪资、确认状态、操作

### 6.3 弹窗 — 工资确认（modal-salary）

- 工资明细 breakdown（基本工资、绩效、社保、公积金、个税等）
- 状态驱动底部按钮：确认无误 / 有异议反馈

---

## 7. 员工管理（department-employees.html）

### 7.1 布局

- 左侧：部门树（可新建部门）
- 右侧：员工列表

### 7.2 员工 Tab

在职员工 / 待转正员工 / 离职员工

### 7.3 列表字段

员工姓名、工号、部门、岗位、入职日期、手机号、员工档案、操作

### 7.4 操作

- 新建员工 / 新建部门
- 编辑 → employee-edit.html
- 查看档案 → employee-profile.html
- 封禁账号（modal-ban）

### 7.5 弹窗

| 弹窗 | 用途 |
|------|------|
| modal-new-dept | 新建部门 |
| modal-ban | 封禁确认 |

---

## 8. 员工档案（employee-profile.html）

- 基本信息、入职信息、合同信息
- 审批/报销/绩效摘要
- 编辑入口 → employee-edit.html
- 侧栏高亮 department-employees

---

## 9. 员工信息编辑（employee-edit.html）

- 表单编辑员工基本信息、岗位、部门
- 保存返回档案页

---

## 10. 模块业务流程

```
员工管理（组织主数据）
  ├─ 员工档案 / 编辑
  ├─ 人事管理（入转离审批）
  ├─ 财务报销（费用审批）
  └─ 薪资管理（工资确认）

经营报表 ← 合同/订单/履约数据汇总
```
