# PRD — 合同订单

| 属性 | 内容 |
|------|------|
| 模块 | 合同订单 |
| 页面 | contract-management.html、order-management.html |
| 优先级 | P0 |
| 版本 | V1.0 |

---

## 1. 模块概述

管理销售成单后的合同签署与订单收款全流程，衔接线索/商机/客户与履约服务。

---

## 2. 共享组件 — 创建合同

| 属性 | 内容 |
|------|------|
| 组件 | assets/create-contract-modal.js |
| 挂载页 | index、customer-management、contract-management、opportunity-management |
| 触发 | `#btn-open-create-contract`、`.js-open-create-contract` |

### 2.1 合同类型 Tab

代理记账 / 注册地址 / 增值服务

### 2.2 核心字段（按类型变化）

- 甲方公司、联系人、联系方式
- 服务套餐、金额、服务周期
- 销售顾问、所属部门

---

## 3. 合同管理（contract-management.html）

### 3.1 筛选

| 字段 | 类型 |
|------|------|
| 合同编号 | 文本 |
| 甲方公司名称 | 文本 |
| 签约状态 | 下拉 |
| 销售顾问 | 下拉 |
| 所属部门 | 下拉 |
| 签订日期 | 区间 |
| 创建时间 | 区间 |

### 3.2 Tab（合同类型）

全部 / 代理记账 / 注册地址 / 增值服务

### 3.3 列表字段

合同编号、甲方公司名称、合同类型、合同金额、签约状态、签订日期、销售顾问、所属部门、创建时间、查看合同、操作

### 3.4 行操作

| 签约状态 | 操作 |
|----------|------|
| 未签 | 导出合同、作废 |
| 已签 | 导出合同 |

### 3.5 页头操作

- 查询 / 重置 / 全部导出
- 创建合同（打开共享弹窗）

## 4. 订单管理（order-management.html）

### 4.1 筛选

| 字段 | 类型 |
|------|------|
| 合同/订单编号 | 文本 |
| 支付状态 | 下拉 |
| 支付渠道 | 下拉 |
| 付款时间 | 下拉 |
| 销售顾问 | 下拉 |
| 所属部门 | 下拉 |

### 4.2 列表字段

订单编号、关联合同、收费单位、订单金额、支付状态、支付渠道、付款时间、是否开票、销售顾问、所属部门、操作

### 4.3 行操作（随支付状态）

| 支付状态 | 操作 |
|----------|------|
| 已支付 | 查看详情、添加订单备注、申请开票 |
| 未支付 | 查看详情、添加订单备注、发送收款码 |

### 4.4 弹窗

| 弹窗 | 用途 |
|------|------|
| modal-payment-qrcode | 发送/展示收款二维码 |
| modal-order-remark | 添加订单备注 |
| modal-order-detail | 订单详情侧栏/弹窗 |
| modal-apply-invoice | 开票申请（批量/单条） |

### 4.5 页头操作

- 查询 / 重置
- 开票申请（批量入口 `#btn-open-apply-invoice`）

### 4.6 订单详情

- 订单基本信息、关联合同、支付记录
- 备注历史
- 开票状态

---

## 5. 业务流程

```
线索/商机/客户
    ↓ 创建合同
合同管理（未签 → 已签）
    ↓ 生成订单
订单管理（未支付 → 已支付）
    ↓
申请开票 / 履约派单（政务/代账/地址）
```
