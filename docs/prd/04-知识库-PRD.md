# PRD — 知识库

| 属性 | 内容 |
|------|------|
| 模块 | 知识库 |
| 页面 | knowledge-base.html |
| 优先级 | P1 |
| 版本 | V1.0 |

---

## 1. 模块概述

提供企业内部文档管理与问答社区，支撑销售、经办人员快速获取操作规范与业务知识。

---

## 2. 用户故事

| 编号 | 角色 | 故事 | 验收 |
|------|------|------|------|
| K-01 | 员工 | 浏览/搜索文档 | 左侧列表切换，右侧预览 |
| K-02 | 管理员 | 上传文档 | 上传按钮可用（原型演示） |
| K-03 | 销售 | 在问答大厅提问 | 发布弹窗提交 |
| K-04 | 员工 | 回答问题 | 回答弹窗，1000字限制 |
| K-05 | 员工 | 点赞有用回答 | 点赞数展示 |

---

## 3. 页面布局

### 3.1 三栏结构

| 区域 | 宽度 | 内容 |
|------|------|------|
| 左栏 | col-span-3 | 文档列表、上传、搜索 |
| 右栏 | col-span-9 | 问答大厅 / 文档预览 |

### 3.2 模式切换

- 默认：问答大厅（qa-panel）
- 点击左侧文档：切换为文档预览（doc-panel），隐藏问答

---

## 4. 文档管理

### 4.1 功能

| ID | 功能 | 描述 |
|----|------|------|
| DOC-01 | 上传文档 | 页头按钮「上传文档」 |
| DOC-02 | 文档搜索 | 左侧关键词搜索框 |
| DOC-03 | 文档列表 | 按类型图标区分 pdf/img/sheet |
| DOC-04 | 文档预览 | 右侧 article 渲染 HTML 内容 |

### 4.2 文档类型

| type | 图标 | 示例 |
|------|------|------|
| pdf | fa-file-pdf | 平台操作手册 |
| img | fa-file-image | 用户开发规范 |
| sheet | fa-file-lines | 指标管理办法 |

### 4.3 内置文档（原型数据）

平台操作手册、用户开发规范、指标管理办法及操作手册进阶/合同/订单/开票/客户/线索篇

---

## 5. 问答大厅

### 5.1 功能

| ID | 功能 | 描述 |
|----|------|------|
| QA-01 | 发布问题 | btn-open-ask → modal-ask-question |
| QA-02 | 关键词检索 | 搜索框 + 检索按钮 |
| QA-03 | 问答列表 | 问题、回答、作者、时间、点赞 |
| QA-04 | 去回答 | js-open-answer → modal-answer-question |
| QA-05 | 展开阅读 | 长回答折叠/展开 |

### 5.2 弹窗字段

| 弹窗 | 字段 | 校验 |
|------|------|------|
| modal-ask-question | 发布问题 | 必填，≤1000字，实时计数 |
| modal-answer-question | 回答内容 | 必填，≤1000字，实时计数 |
