# PRD — 登录与工作台

| 属性 | 内容 |
|------|------|
| 模块 | 入口 / 工作台 |
| 页面 | login.html、index.html |
| 优先级 | P0 |
| 版本 | V1.0 |

---

## 1. 模块概述

提供系统入口认证与登录后首页驾驶舱，聚合高频操作、KPI、待办与业绩排行。

---

## 2. 用户故事

| 编号 | 角色 | 故事 | 验收 |
|------|------|------|------|
| L-01 | 员工 | 用工号密码登录系统 | 登录成功进入工作台 |
| W-01 | 销售 | 快速录入线索/跟进 | 弹窗完成或 toast 提示 |
| W-02 | 销售 | 发起合同签署 | 打开创建合同侧栏 |
| W-03 | 管理者 | 查看本月 KPI 与排行 | 数据概览/工作简报/排行可见 |
| W-04 | 全员 | 查看待办与知识库摘要 | 侧栏卡片可跳转 |

---

## 3. 功能清单 — login.html

### 3.1 页面信息

- 标题：企税康 - 员工登录
- 路由：login.html → index.html

### 3.2 功能需求

| ID | 功能 | 描述 | 规则 |
|----|------|------|------|
| LOGIN-01 | 密码登录 | 工号 + 密码 | 必填 |
| LOGIN-02 | 显示密码 | 切换明文/密文 | 图标切换 |
| LOGIN-03 | 提交登录 | 跳转工作台 | 当前原型无真实鉴权 |

### 3.3 字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 工号 | 文本 | 是 | name=uid |
| 密码 | 密码 | 是 | name=pwd |

---

## 4. 功能清单 — index.html（工作台）

### 4.1 页面信息

- 标题：工作台
- 面包屑：首页 / 工作台

### 4.2 功能区块

#### A. 欢迎横幅

- 展示用户问候、日期

#### B. 快捷入口

| 入口 | 行为 |
|------|------|
| 快速录入线索 | 打开 modal-add-lead |
| 新增跟进说明 | 打开 modal-follow-note |
| 发起合同签署 | 打开 modal-create-contract |
| 添加客户维护 | 打开 modal-maintain |

#### C. 数据概览

- 筛选：园区、人员、周期（本月）
- 指标：业绩进度、户数进度、续费进度、线索转化（环形进度 + 数值）

#### D. 工作简报

- 筛选：2×全部 + 本月
- 指标（10项）：新增线索、新增商机、跟进中、合同审核、合同待签、订单未支付、工商未派单、代账未派单、代账到期、地址到期

#### E. 业务速览

- 表格：编号、业务类型、服务项目、办理状态、经办人、销售员
- 外链：government-affairs.html「查看全部」

#### F. 平台知识库

- 文档列表摘要
- 外链：knowledge-base.html

#### G. 代办事项

- 待办列表（合同、审批、回访、资料、续费等）
- 行内「去处理」链接

#### H. 月度业绩排行

- 月份选择
- Tab：个人排行 / 团队排行
- 维度：业绩金额 / 完成率
- Top3 领奖台 + 4-5名列表
- 外链：sales-performance.html

### 4.3 弹窗需求

| 弹窗 ID | 名称 | 关键字段 |
|---------|------|----------|
| modal-add-lead | 新增线索 | 姓名≤5、手机≤11、区域、来源、备注≤30 |
| modal-follow-note | 添加跟进 | 线索/商机*、跟进内容≤1000 |
| modal-maintain | 添加维护 | 客户、维护类型、内容 |
| modal-create-contract | 创建合同 | 共用 create-contract-modal.js |

### 4.4 交互规则

- Esc 关闭弹窗
- 字符计数实时更新
- 保存成功 toast（原型无持久化）
