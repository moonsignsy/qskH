## 3. 销售目标（sales-target.html）

### 3.1 功能概述

按部门/人员设置月度销售目标，支持批量设置与单条编辑。

### 3.2 筛选

| 字段 | 类型 |
|------|------|
| 目标周期 | month（单月） |
| 所属部门 | 下拉 |
| 设置状态 | 全部/已设置/待设置 |
| 关键词 | 部门/姓名 |

### 3.3 Tab

- 部门目标 / 人员目标

### 3.4 列表字段

**部门：** 勾选、部门、负责人、人数、业绩目标、每月新增户数、已完成、完成率、状态、操作

**人员：** 勾选、姓名、部门、岗位、业绩目标、每月新增户数、已完成、完成率、状态、操作

### 3.5 操作

| 操作 | 说明 |
|------|------|
| 导入模板 / 导出 | 页头 |
| 批量设置目标 | 勾选后弹窗 |
| 编辑/设置 | 行操作 |

### 3.6 弹窗 — 批量设置

- 目标周期（month）*
- 业绩目标（元）*
- 每月新增户数
- 备注
- 覆盖已有目标（checkbox，未勾选仅更新「待设置」）

### 3.7 业务规则

- 业绩目标必填
- 全选/半选联动
- 概览卡：本月已设目标、业绩合计、待设置、平均完成率

## 4. 线索管理（lead-management.html）

### 4.1 筛选

客户姓名/手机、是否有效、所属区域、企业类型、来源渠道、创建时间区间

### 4.2 Tab 状态

全部 / 跟进中 / 已成单 / 已放弃（`data-tab-cat` 过滤）

### 4.3 列表字段

勾选、客户姓名、手机、区域、行业、企业类型、来源、跟进结果、线索有效、创建时间、销售顾问、最后跟进、跟进日志、操作

### 4.4 行操作（随状态变化）

| 状态 | 操作 |
|------|------|
| 待跟进 | 首次跟进 |
| 跟进中 | 编辑客户信息、催跟进、添加跟进、转商机、生成报价单 |
| 已成单 | 编辑 |
| 已放弃 | 无 |

### 4.5 弹窗

| 弹窗 | 用途 |
|------|------|
| modal-new-lead | 新增线索 |
| modal-transfer | 移交 |
| modal-first-follow | 首次跟进（有效/无效分支） |
| modal-follow | 添加跟进 |
| modal-supplement | 补充客户信息 |
| modal-quote | 报价单（代账/地址/增值模块） |
| modal-convert-opp | 转商机（阶段+等级） |

### 4.6 字段校验

- 姓名≤5、手机≤11、备注≤30、跟进≤1000
- 转商机必选阶段、等级

### 4.7 外链

→ lead-follow-detail.html?id=lead-{n}

## 5. 跟进日志详情（lead-follow-detail.html）

### 5.1 布局

- 左：跟进进度时间轴（创建→首次跟进→跟进→报价→合同→支付→成单/放弃）
- 右：线索信息 + 内嵌添加跟进

### 5.2 功能

- URL `?id=lead-1~6` 驱动数据
- 上一条/下一条导航
- 报价单预览弹窗
- 外链 contract-management.html

---

## 6. 商机管理（opportunity-management.html）

### 6.1 筛选

客户名称、联系人/联系方式、销售顾问、商机阶段、创建时间

### 6.2 列表字段

客户名称、联系人、联系方式、销售顾问、商机等级、商机阶段、创建时间、查看详情、操作

### 6.3 行操作

| 阶段 | 操作 |
|------|------|
| 意向/部分成交 | 新增合同、客户信息、添加跟进 |
| 成交 | 客户信息 |
| 流失 | 无 |

### 6.4 弹窗

- modal-supplement：客户信息
- modal-follow：添加跟进（流失阶段显示流失说明）

### 6.5 外链

→ opportunity-detail.html?id=opp-{n}

---

## 7. 商机详情（opportunity-detail.html）

- 跟进说明时间轴 + 商机信息网格
- 内嵌添加跟进（阶段/时间/内容/流失说明）
- `?id=opp-1~5` 驱动
- 上一条/下一条

---

## 8. 销售业绩（sales-performance.html）

### 8.1 筛选

统计周期（month）、所属部门、岗位类型、关键词

### 8.2 Tab 结构

**一级：** 个人排名 / 部门排名 / 业绩明细

**二级 pill：** 销售员|顾问员、销售部门|顾问部门、个人|部门业绩

**排行维度：** 业绩金额 / 完成率 / 新增户数

### 8.3 展示

- 4 张概览卡
- Top3 领奖台
- 排名表格（含环比、完成率进度条）
- 业绩明细含总计行

---

## 9. 模块级业务流程

```
线索(lead) --转商机--> 商机(opp) --新增合同--> 合同
     |                      |
     v                      v
跟进详情              商机详情
     \____________________/
              |
              v
        销售目标 ←→ 销售业绩（考核）
```
