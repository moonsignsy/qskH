# PRD — 系统管理

| 属性 | 内容 |
|------|------|
| 模块 | 系统管理 |
| 页面 | role-permissions、dictionary-management、menu-management |
| 优先级 | P1 |
| 版本 | V1.0 |

---

## 1. 模块概述

提供角色权限、数据字典与菜单配置能力，支撑平台可配置化与多角色访问控制。

---

## 2. 角色权限（role-permissions.html）

### 2.1 布局

- 左侧：角色列表（搜索、新增）
- 右侧：权限配置面板

### 2.2 Tab

角色权限 / 员工权限

### 2.3 角色列表

| 功能 | 说明 |
|------|------|
| 搜索角色 | 按角色名称 |
| 新增角色 | modal-add-role |
| 选择角色 | 右侧加载权限树 |

### 2.4 权限树结构

按模块分组 checkbox 树，覆盖：

- 工作台
- 销售管理（目标、线索、商机、业绩）
- 客户中心（客户、政务、代账、地址、物品）
- 知识库
- 合同订单
- 运营管理（报销、报表、人事、薪资、员工）
- 系统管理（角色权限、字典、菜单）

> 说明：已移除「开票管理」独立权限块（对应页面已删除）。

### 2.5 员工权限 Tab

- 按员工搜索
- 展示员工已分配角色与权限摘要
- 支持单独授权调整

### 2.6 弹窗 — 新增角色

| 字段 | 必填 |
|------|------|
| 角色名称 | 是 |
| 角色描述 | 否 |

---

## 3. 字典管理（dictionary-management.html）

### 3.1 布局

- 左侧：字典分类树
- 右侧：字典项列表

### 3.2 字典分类（示例）

线索来源、企业类型、业务类型、报销类型、申请类型等

### 3.3 列表字段

字典编码、字典名称、排序、状态、操作

### 3.4 操作

- 新增字典项（modal-dict-add）
- 编辑
- 删除（modal-dict-delete 确认）

### 3.5 弹窗字段

| 字段 | 说明 |
|------|------|
| 字典编码 | 唯一标识 |
| 字典名称 | 显示名称 |
| 排序 | 数字 |
| 启用状态 | 开关 |

---

## 4. 菜单管理（menu-management.html）

### 4.1 功能概述

可视化配置侧栏菜单结构，与 `assets/menu-data.js` DEFAULT_MENUS 对齐，保存至 localStorage。

### 4.2 概览卡

菜单总数、已启用、已禁用、分组数

### 4.3 菜单列表

| 列 | 说明 |
|----|------|
| 菜单名称 | 一级/子菜单 |
| 类型 | link / group |
| 路由 | href |
| 图标 | Font Awesome class |
| 排序 | 行内上移/下移 |
| 启用 | 开关 |
| 操作 | 编辑 |

### 4.4 侧栏预览

- 实时预览已启用菜单结构
- 保存后全站 app-shell 读取 MenuData.getMenus()

### 4.5 弹窗 — 编辑菜单（modal-menu-edit）

| 字段 | 说明 |
|------|------|
| 菜单名称 | 必填 |
| 路由地址 | 一级/分组必填；子菜单可空 |
| 图标 | FA 类名 |
| 启用该菜单 | checkbox |

### 4.6 操作

| 操作 | 说明 |
|------|------|
| 保存配置 | 写入 localStorage |
| 恢复默认 | 清除 STORAGE_KEY，重置 DEFAULT_MENUS |
| 搜索 | 按名称或路由过滤 |

### 4.7 默认菜单结构（7 模块）

1. 工作台
2. 销售管理（4 子项）
3. 客户中心（5 子项）
4. 知识库
5. 合同订单（2 子项）
6. 运营管理（5 子项，含人事管理）
7. 系统管理（3 子项）

---

## 5. 模块依赖关系

```
字典管理 → 各业务页下拉选项数据源
角色权限 → 菜单可见性 + 操作按钮控制
菜单管理 → app-shell 侧栏渲染
```
